PLAN DE IMPLEMENTACIÓN - APP ANDROID v2.0 PARA CONTROL DE LEDS
==============================================================

SISTEMA: Android 12+ con One UI
VERSIÓN: 2.0 - Control de Brillo y Dirección

NUEVAS CARACTERÍSTICAS v2.0:
============================
- Control de brillo con slider
- Selección de dirección de luces
- Switch para activar/desactivar modo sonido
- Interfaz optimizada para control avanzado

PASOS DE IMPLEMENTACIÓN v2.0:
=============================

1. CONFIGURACIÓN DEL DISPOSITIVO:
   - Mismo proceso que versión 1.0
   - Activar modo desarrollador
   - Habilitar depuración USB

2. CONFIGURACIÓN DE ANDROID STUDIO:
   - Crear nuevo proyecto: "Empty Activity"
   - Nombre: "LED Controller v2.0"
   - Package: com.example.ledcontrollerv2
   - API mínima: 21 (Android 5.0)

3. CONFIGURACIÓN DEL PROYECTO v2.0:
   - Copiar MainActivity.java desde: /home/mmorana/IA/valen/2.0/android/led_controller_v2/MainActivity.java
     al directorio: app/src/main/java/com/example/ledcontrollerv2/
   - Copiar activity_main.xml desde: /home/mmorana/IA/valen/2.0/android/led_controller_v2/activity_main.xml
     al directorio: app/src/main/res/layout/
   - Reemplazar AndroidManifest.xml con: /home/mmorana/IA/valen/2.0/android/led_controller_v2/AndroidManifest.xml
     en: app/src/main/

4. CONFIGURACIÓN BLUETOOTH:
   - Obtener MAC address del HC-05
   - Actualizar DEVICE_ADDRESS en MainActivity.java
   - Línea 17: private static final String DEVICE_ADDRESS = "TU_MAC_AQUI";

5. COMPILACIÓN E INSTALACIÓN:
   - Build → Make Project
   - Run → Run 'app'
   - Instalar en Samsung Z Fold4

6. CONFIGURACIÓN DE PERMISOS:
   - Permitir acceso a Bluetooth
   - Permitir acceso a ubicación

7. EMPAREJAMIENTO:
   - Buscar "LEDController_v2" o "HC-05"
   - PIN: 1234

8. PRUEBAS v2.0:
   - Conectar Bluetooth
   - Probar switch de modo sonido
   - Ajustar brillo con slider
   - Probar botones de dirección

CONTROLES DE LA APP v2.0:
=========================

MODO SONIDO:
- Switch ON/OFF para activar respuesta a sonido
- Cuando está ON, LEDs reaccionan automáticamente

CONTROL DE BRILLO:
- Slider de 0 a 255
- Cambio en tiempo real
- Afecta intensidad general

DIRECCIÓN DE LUCES:
- "Todas": Toda la tira se ilumina
- "Izquierda": Solo mitad izquierda
- "Derecha": Solo mitad derecha  
- "Centro": Solo la parte central

OPTIMIZACIÓN PARA Z FOLD4 v2.0:
===============================
- Interfaz adaptada a pantallas grandes
- Controles más espaciados para pantalla plegable
- Funciona en ambas pantallas (interna/externa)

COMANDOS ENVIADOS v2.0:
=======================
- "SOUND_ON" / "SOUND_OFF"
- "BRIGHTNESS:0-255"
- "DIRECTION:0-3"

TROUBLESHOOTING v2.0:
====================
- Brillo no cambia: verificar conexión Bluetooth
- Dirección no funciona: verificar que NUM_LEDS sea correcto
- Modo sonido no responde: verificar micrófono en Arduino
