PASO A PASO - IMPLEMENTACIÓN ANDROID 2.0
=========================================

PASO 1: PREPARAR DISPOSITIVO
- Ir a Configuración → Acerca del teléfono
- Tocar 7 veces "Número de compilación"
- Activar "Opciones de desarrollador"
- Habilitar "Depuración USB"

PASO 2: CREAR PROYECTO
- Abrir Android Studio
- New Project → Empty Activity
- Nombre: "LED Controller v2.0"
- Package: com.example.ledcontrollerv2
- API mínima: 21

PASO 3: COPIAR ARCHIVOS
Copiar estos archivos a tu proyecto:

MainActivity.java → app/src/main/java/com/example/ledcontrollerv2/
Desde: /home/mmorana/IA/valen/2.0/android/led_controller_v2/MainActivity.java

activity_main.xml → app/src/main/res/layout/
Desde: /home/mmorana/IA/valen/2.0/android/led_controller_v2/activity_main.xml

AndroidManifest.xml → app/src/main/
Desde: /home/mmorana/IA/valen/2.0/android/led_controller_v2/AndroidManifest.xml

PASO 4: OBTENER MAC DEL HC-05
- En tu Samsung Z Fold4: Configuración → Bluetooth
- Buscar "LEDController_v2"
- Emparejar (PIN: 1234)
- Anotar dirección MAC (XX:XX:XX:XX:XX:XX)

PASO 5: ACTUALIZAR CÓDIGO
- Abrir MainActivity.java
- Línea 17: Cambiar DEVICE_ADDRESS por tu MAC
- Ejemplo: "98:D3:31:FB:48:E2"

PASO 6: COMPILAR E INSTALAR
- Conectar Z Fold4 por USB
- Build → Make Project
- Run → Run 'app'
- Seleccionar tu dispositivo
- Esperar instalación

PASO 7: CONFIGURAR PERMISOS
- Abrir app instalada
- Permitir acceso a Bluetooth
- Permitir acceso a ubicación

PASO 8: PROBAR CONTROLES
- Tocar "Conectar Bluetooth"
- Verificar mensaje "Conectado"
- Probar switch "Modo Sonido"
- Mover slider de brillo
- Probar botones de dirección:
  * Todas
  * Izquierda  
  * Derecha
  * Centro

¡ANDROID 2.0 LISTO!

CONTROLES DISPONIBLES:
- Switch: Activar/desactivar respuesta a sonido
- Slider: Ajustar brillo (0-255)
- Botones: Dirigir luces a diferentes zonas
